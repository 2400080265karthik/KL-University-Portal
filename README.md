# KL University Student Portal

**Student ID:** 2400080265

## Project Overview

This is a responsive web-based student portal for KL University, designed to help students manage their academic projects and track their progress. The portal provides a clean, modern interface with navigation to various student resources.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional interface with smooth transitions
- **Quick Navigation**: Easy access to dashboard, courses, grades, and projects
- **Student Information**: Displays student ID and university name
- **Professional Styling**: Teal color scheme with consistent typography

## File Structure

```
KL-University-Portal/
├── index.html      # Main HTML file with portal structure
├── styles.css      # CSS styling and responsive design
└── README.md       # Project documentation
```

## Technologies Used

- **HTML5**: Semantic markup for structured content
- **CSS3**: Modern styling with flexbox layout and media queries
- **Responsive Design**: Mobile-first approach

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/2400080265karthik/KL-University-Portal.git
   ```

2. Navigate to the project directory:
   ```bash
   cd KL-University-Portal
   ```

3. Open `index.html` in your web browser to view the portal

## Customization

### Colors
Modify the primary color in `styles.css`:
- Primary: `#1a5f7a` (Teal)
- Secondary: `#2a7a99` (Lighter Teal)
- Accent: `#ffc107` (Gold)

### Content
Edit `index.html` to add:
- Your institution logo
- Additional sections
- More navigation items
